面向对象：
    final 最终类，不能被继承，只能实例化（创建对象）

    abstract 抽象类能被继承，不能被实例化
      abctract function move();  抽象方法，只能放在抽象类里
      不同的类继承同一个抽象类，抽象类的抽象方法同一个方法可以实现多种动作。  这种叫多态

    interface 接口（接口不是类，是类的一种规范）
        接口里边只能有类常量,接口的抽象方法不需要关键字abstract
          interface  anmial{ const PI=3.14;}
          接口作用到类上：   class 类名  implements  接口名{}
          接口中的方法都得在类中实现可以写空方法
    方法重载
      处理不可访问的一种方法的一种机制，利用两个魔术方法来执行
      __call($name,$array)    //调用不可访问的非静态方法   （提示，默认等可以应用）
      __callstaic()  //调用不可访问的静态方法

      self  代码预编译的时候指定
      static 代码执行的时候绑定   static::类内函数或者变量
  单例模式是一种设计模式（有截图）
      实现控制无线创建对象  俗称三私（私有属性 私有构造，私有克隆 ）一公（公用静态接口（函数） ）

      帮其他类创建对象（需要引入对应的类文件）
